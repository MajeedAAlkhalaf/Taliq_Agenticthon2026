# Taliq_Agenticthon2026

Taliq is an AI powered English language learning application made for Saudi university students. Unlike generic platforms such as Duolingo or Rosetta Stone, Taliq bridges the gap between everyday English and the highly specific academic language required by Saudi higher education institutions. In addition, our agent "Zaki" customizes the learning experience and difficulty to suit the progression of the student. 

This repo is made for the 2026 Agenticthon in collaboration with Ziyad Islam, Ahmed Alshaikh and Ali Farhan.
